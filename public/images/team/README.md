## ✅ So legst du die Fotos ab:

1. Fotos umbenennen nach Schema oben (lowercase, mit bindestrichen)
2. In diesen Ordner kopieren: `public/images/team/`
3. Fertig! Website zeigt die Fotos automatisch an

**Beispiel:**
```
public/images/team/
├── rayan-tawfik.jpg      ← Foto von Rayan
├── konrad-schreiner.jpg  ← Foto von Konrad
```
